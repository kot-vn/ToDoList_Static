# ToDoList_Static

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
cd dist
git init
git add -A
git commit -m "deploy"
git push -f https://github.com/kot-vn/ToDoList_Static.git master:gh-pages
```
